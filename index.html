Vidu 剧本批量生成 Demo（角色图 -> 参考生视频）

解析剧本 JSON 后并发创建任务：先批量角色图，再批量场景视频；支持 audio / auto_subjects / audio_type / model 动态设置。

运行配置

鉴权头使用 Authorization: Token {api_key}。 图片任务默认仅使用 appearance_prompt，可手动开启拼接 voice_prompt。 视频任务使用 subjects + visual_prompt，场景里的 @角色名 会按主体名引用。 部署到 GitHub Pages 时，请将代理地址配置为你的 Cloudflare Worker 域名。
